windows_app_adbpush
===================
ドラッグ＆ドロップでファイルをpushするアプリです。  
複数ファイルのドロップに対応してます。  
権限を空白にすると権限を変更しません。  
apkのインストール/アップデートが出来ます。  
アプリ起動時にadb start-serverを実行します。  
デフォルトは10秒でタイムアウトするので、大きなファイルの転送時は10秒のチェックボックスを外してください。  
コマンドとログのテキストボックスをダブルクリックすると消去します。  
1ファイルだけならEXEに直接ドロップすることが出来ます。（/data/local/tmpに転送します）
